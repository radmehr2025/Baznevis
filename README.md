**بازنویس** یک ابزار تحت وب سبک، هوشمند و تعاملی است که برای اصلاح نگارش فارسی، با تمرکز ویژه بر استفاده صحیح از **نیم‌فاصله (ZWNJ)** طراحی شده است.

## 🚀 قابلیت‌ها
- **اصلاح هوشمند نیم‌فاصله:** تشخیص و اصلاح خودکار پیشوندها و پسوندها.
- **دیتابیس کلمات مرکب:** بهره‌مندی از یک لغت‌نامه داخلی برای اصلاح کلمات پیچیده (مانند «نرم‌افزار»).
- **ویرایش تعاملی:** با کلیک روی کلمات اصلاح شده در خروجی، می‌توانید نیم‌فاصله را به فاصله معمولی تبدیل کنید.
- **اصلاح علائم نگارشی:** تنظیم فاصله صحیح علائمی مانند ویرگول و علامت سوال.
- **فارسی‌سازی اعداد:** تبدیل خودکار اعداد انگلیسی به معادل فارسی.
- **کاملاً رسپانسیو:** طراحی بهینه شده برای نمایش صحیح در دسکتاپ، تبلت و موبایل.

## 🛠️ تکنولوژی‌ها
- HTML5, CSS3, Vanilla JavaScript.

## 📖 راهنمای استفاده
متن خود را وارد کرده، روی دکمه «شروع اصلاح» کلیک کنید، در صورت نیاز با کلیک روی کلمات آن‌ها را ویرایش دستی کنید و در نهایت متن را کپی نمایید.

## 🤝 مشارکت
دیتابیس کلمات مرکب این پروژه به مرور زمان کامل‌تر خواهد شد. برای افزودن کلمات جدید، خوشحال می‌شویم Pull Request ارسال کنید!
