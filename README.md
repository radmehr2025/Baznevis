# ✍️ بازنویس (ویرایشگر هوشمند متن فارسی)

**بازنویس** یک ابزار تحت وب سبک، هوشمند و تعاملی است که برای اصلاح نگارش فارسی، با تمرکز ویژه بر استفاده صحیح از **نیم‌فاصله (ZWNJ)** طراحی شده است.

[🇺🇸 View English Version](README-en.md)

## 🚀 قابلیت‌ها
- **اصلاح هوشمند نیم‌فاصله:** تشخیص و اصلاح خودکار پیشوندها (مانند «می»، «نمی») و پسوندها (مانند «ها»، «تر»، «ترین»).
- **دیتابیس کلمات مرکب:** بهره‌مندی از یک لغت‌نامه داخلی برای اصلاح کلمات پیچیده (مانند «نرم‌افزار»، «گفت‌وگو»).
- **ویرایش تعاملی:** قابلیت منحصر به فرد! در متن خروجی، روی هر کلمه‌ای که نیم‌فاصله دارد کلیک کنید تا به سادگی به حالت فاصله معمولی برگردد.
- **اصلاح علائم نگارشی:** تنظیم فاصله صحیح علائمی مانند ویرگول، نقطه‌ویرگول و علامت سوال.
- **فارسی‌سازی اعداد:** تبدیل خودکار اعداد انگلیسی به معادل فارسی.
- **کاملاً رسپانسیو:** طراحی بهینه شده برای نمایش صحیح در دسکتاپ، تبلت و موبایل.

## 🛠️ تکنولوژی‌های مورد استفاده
- HTML5
- CSS3 (Flexbox, Media Queries)
- Vanilla JavaScript (بدون وابستگی به کتابخانه‌های جانبی)

## 📖 راهنمای استفاده
۱. متن فارسی خود را در کادر «متن ورودی» وارد کنید.
۲. روی دکمه **«🔍 شروع اصلاح متن»** کلیک کنید.
۳. نتیجه را در کادر مقابل مشاهده کنید. اگر کلمه‌ای به اشتباه اصلاح شده، کافیست روی آن کلیک کنید.
۴. با دکمه **«📋 کپی متن نهایی»**، متن اصلاح شده را کپی کنید.
